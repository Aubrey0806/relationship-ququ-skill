# 曲曲大女人 Perspective Skill

A Claude Code skill that distills the cognitive framework of **曲曲大女人** (乐传曲) — China's most controversial relationship strategist — into a reusable AI perspective.

## Core Concept

Rather than mimicking how 曲曲 speaks, this skill extracts her underlying "cognitive operating system" — the mental models, decision heuristics, and thinking patterns behind her radical view that **all relationships are value exchanges**.

## Key Features

- **4 core mental models**: Value Exchange Theory, 雄竞/雌竞 Framework, Asset-Minded Thinking, Risk-Averse Rationalism
- **7 decision heuristics** distilled from 1,000+ hours of livestreams and courses
- **Expression DNA**: her signature de-moralized, business-language style
- **Honest limitations**: including the irony of her own 758万 tax evasion case
- **Complete research**: 6 source documents covering writings, conversations, external views, decisions, and timeline

## Installation

```bash
npx skills add Aubrey0806/relationship-ququ-skill
```

Then activate by saying: **「用曲曲的视角分析一下...」**

## What This Skill Does

Use this perspective to:
- Analyze the **value structure** of a relationship (who's gaining, who's losing)
- Identify **value mismatches** before you over-invest emotionally
- Apply **借力打力** thinking to social and professional relationships
- Understand why "love" and "exchange" aren't mutually exclusive

## What This Skill Doesn't Do

- Handle purely emotional questions without strategic dimensions
- Provide legal, financial, or psychological advice
- Endorse tax evasion, manipulation, or deception
